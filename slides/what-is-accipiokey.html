What is Accipio Key?

  • Proof of concept
  • Improves user typing experience
  • Draws inspiration from current smartphone technologies
Accipio Key is a proof of concept assistive application which aims to help users type in an easier, faster and less frustrating way by utilizing full-text search technologies to normalize and streamline the typing experience on desktops. This project draws inspiration from current smartphone technologies and it tries to mimic the most useful features as best as possible.

How Does Accipio Key Work?

  1. Runs in the background
  2. Listens for user input
  3. Analyzes input (identifies misspellings, typos, snippets)
  4. Takes appropriate action (correction, completion, generation)
Accipio Key does this by running in the background, listening for user keyboard input and providing text correction, completion, and generation across all applications in a standardized manner. This set of standardized features allow users to quickly and effortlessly navigate between applications and retain the word processing features they are accustomed to use.
